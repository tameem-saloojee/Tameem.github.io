  • The differences between html and css
  • First off, HTML and CSS aren’t the same. They function as a team! Together, they form the bones and skin for any website. HTML stands for Hypertext Mark-up Language. Think of HTML as the skeleton of the document. The HTML is what gives structure to the site. This is done through tags, elements, and attributes. Whether you want headings, lists, images, or links, HTML can do all of that. CSS stands for Cascading Style Sheet. Without CSS, your websites would look rather boring, dull, and pretty much naked. In CSS, there is a property and a value. Property is the quality you want to change; value is amount of change. We can augment the HTML body with CSS by adding details. For example, let’s say we want to change the colour of the body or Colour and size of any text we put in the body. This is all done through CSS

  • What are Functions ?
  • Functions are one of the fundamental building blocks in JavaScript. A function is a JavaScript procedure—a set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call it. This eliminates the need of writing the same code again and again. It helps programmers in writing modular codes. Functions allow a programmer to divide a big program into a number of small and manageable functions.
